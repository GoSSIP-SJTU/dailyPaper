G.O.S.S.I.P 学术论文推荐 2020-05-06

RomanGol 上海交大软件安全研究组GoSSIP
5月6日

美好的5月到了！本月的第一篇学术论文，我们继续关注PLDI 2020上关于binary code的研究工作。今天介绍的研究论文来自UIUC的研究人员，聚焦binary code lifter的实现正确性



binary code lifting指的是将二进制代码翻译为更高级的中间语言（IR）的过程，这一过程受到二进制代码中信息缺失的限制，往往存在很多不准确性甚至错误。然而在现实世界中，诸多的binary code lifter都没有经过严格的测试。作者在本文中聚焦于McSema（https://github.com/lifting-bits/mcsema，还记得我们之前推荐的BinRec就是和这个工具比较的吗？）工具的准确性，该工具是目前将x86-64二进制代码翻译成LLVM IR的代表性工作，然而作者通过严格的语义等价测试（semantic equivalence test）发现了McSema中存在的针对29条指令的翻译问题





作者的另一个发现是提出了针对程序的语义等价的非常高效的算法：作者指出，如果需要检查函数F和F'的语义等价性，使用符号执行进行求解和比较会带来巨大的性能开销，而作者首先用LLVM的优化pass对F和F'进行变换，然后再对优化后函数的数据依赖图（data dependence graph）进行图同构比较（graph-isomorphism check），这样会极大提升比较性能！
