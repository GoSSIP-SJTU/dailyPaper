G.O.S.S.I.P 学术论文推荐 2020-07-03

mmm 上海交大软件安全研究组GoSSIP
1周前

今天我们和大家分享一篇发表在PLDI’20 学术会议上的论文 Understanding Memory and Thread Safety Practices and Issues in Real-World Rust Programs，论文主要探讨了实践中的Rust程序员 对unsafe code的使用，以及Rust程序中 memory-safety 和 concurrency safety的问题。



作者手动检查了现实Rust程序中850处 unsafe code的使用和170个bug




基于此，作者给Rust开发者和语言开发人员提出了11条感悟和8点建议，关于unsafe code 的使用，作者研究了程序员使用unsafe的原因，移除一些不必要unsafe code的方式，以及封装interior unsafe的应用。



对于memory safety bug，作者根据bug的本身的性质，以及bug的根源和效果是否在unsafe/safe code将bug进行了分类。作者相应地提出了几条修复的建议。



对于 thread-safety bug，作者也根据 synchronization 和 communication 的方式进行了归类，并提出了一些修复建议。




最后，作者开发了两款静态分析的工具，用于检测 use-after- free and double-lock bugs ，并为之后的研究人员提出了了多条如何开发Rust漏洞检测工具的建议。
