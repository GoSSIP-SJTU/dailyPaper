#安全学术论文每日推荐
20个

作为圣诞节大礼包，今天送上ASPLOS 2021预录取论文列表中和安全研究相关的论文，希望大家能开心过节，快乐科研！



ASPLOS作为顶级的计算机系统结构和程序语言相关的学术会议，2021年将在线举办，目前会议将拟录取的论文的长摘要（extended abstract）全部放出，可以在下面网址查询：

https://asplos-conference.org/papers/


===========================

Who’s Debugging the Debuggers? Exposing Debug Information Bugs in Optimized Binaries

https://asplos-conference.org/abstracts/asplos21-paper58-extended_abstract.pdf


这篇文章讨论了编译器在生成调试版本的可执行文件时，在嵌入调试信息（debug information）过程中可能出现的bug，并提出了相关分析框架来发现此类问题


===========================

C11Tester: A Fuzzer for C/C++ Atomics

https://asplos-conference.org/abstracts/asplos21-paper172-extended_abstract.pdf


又是一篇fuzzing论文，专门针对C++11标准引入的low-level atomics operations可能产生的并发bug的fuzzing方法



===========================

CubicleOS: A Library OS with Software Componentisation for Practical Isolation

https://asplos-conference.org/abstracts/asplos21-paper486-extended_abstract.pdf


和容器相关的论文，基于Intel MPK技术，实现了名为CubicleOS的系统，对不同的library OS组件进行隔离



===========================

Speculative Interference Attacks: Breaking Invisible Speculation Schemes

https://asplos-conference.org/abstracts/asplos21-paper148-extended_abstract.pdf


论文提出了speculative interference attacks，这类攻击在针对目前一些部署了名为invisible speculation的防御的系统上仍然可以工作



===========================

Jamais Vu: Thwarting Microarchitectural Replay Attacks

https://asplos-conference.org/abstracts/asplos21-paper255-extended_abstract.pdf


论文针对2019年ISCA上提出的Microarchitectural Replay Attack进行了防御


===========================

PIBE: Practical Kernel Control-flow Hardening with Profile-guided Indirect Branch Elimination

https://asplos-conference.org/abstracts/asplos21-paper701-extended_abstract.pdf


顾名思义，论文使用了profiling来帮忙，进行控制流跳转的分析，对抗Kernel中的控制流劫持攻击


===========================

Automatically Detecting and Fixing Concurrency Bugs in Go Software Systems

https://asplos-conference.org/abstracts/asplos21-paper1170-extended_abstract.pdf


一篇关于Go语言中并发bug的自动化检测分析论文，支持对发现的bug进行修复


===========================

Incremental CFG Patching for Binary Rewriting

https://asplos-conference.org/abstracts/asplos21-paper1687-extended_abstract.pdf


作者设计了一种名为Trampoline Placement Analysis的技术，在二进制代码中经济地插入相关控制代码，实现二进制重写和修复，取得分析和效率的平衡


===========================

In-Fat Pointer: Hardware-Assisted Tagged-Pointer Spatial Memory Safety Defense with Subobject Bound Granularity Protection

https://asplos-conference.org/abstracts/asplos21-paper1439-extended_abstract.pdf


又是一篇在pointer实现上进行安全增强的研究论文，作者提出了In-Fat Pointer技术，在RISC-V 64位指令集上进行了仿真实现


===========================

祝大家在2020的最后一星期也能继续学习进步！
