今天我们转到ISSTA 2020学术会议，去看看一篇关于二进制代码分析的论文Identifying Java Calls in Native Code via Binary Scanning



论文关注的是Android应用中native code的分析问题，在native code中，通过Java Native Interface (JNI) 回调Java字节码的函数是一种比较常见的编程模式，但是在针对native code的二进制代码分析中，如何建立藉由JNI从native code到bytecode的控制流连接，往往没有得到程序分析工具的足够支持。作者针对这一问题开展了研究，实现了针对此类分析的soundness
