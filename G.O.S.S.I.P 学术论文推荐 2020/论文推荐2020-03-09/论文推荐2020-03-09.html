G.O.S.S.I.P 学术论文推荐 2020-03-09

RomanGol 上海交大软件安全研究组GoSSIP
3月9日

今天给大家推荐一篇NDSS 2020学术会议上的论文FUSE: Finding File Upload Bugs via Penetration Testing，在该论文中，作者设计了一个名为FUSE的渗透测试工具，用来发现服务器PHP程序中UFU和UEFU两大类文件上传漏洞（Unrestricted File Upload (UFU) vulnerability，a critical security threat that enables an adversary to upload her choice of a forged file to a target web server. This bug evolves into an Unrestricted Executable File Upload (UEFU) vulnerability）






作者提出了13种mutation策略，对HTTP(S) multipart request的fields进行变换之后测试服务器PHP程序。对33个PHP程序进行测试后，作者一共发现了15个相关的CVE
