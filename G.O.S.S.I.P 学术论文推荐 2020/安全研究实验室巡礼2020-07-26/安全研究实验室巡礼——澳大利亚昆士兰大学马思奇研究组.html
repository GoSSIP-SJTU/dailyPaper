#安全实验室巡礼
20个

本周的安全研究实验室巡礼，我们将为大家介绍来自南半球的澳大利亚昆士兰大学的马思奇助理教授（博导）研究组。



马思奇博士（Dr. Siqi Ma）于2020年加入昆士兰大学信息技术与电子工程学院任助理教授。马老师致力于利用软件工程的技术方法进行软件的安全性分析和防御，从而对真实软件设计和实现进行漏洞检测与分析。同时确定漏洞的类型和造成原因。马老师在安全学术顶级会议和期刊发表的论文中包含了对各类软件系统的分析，其中包括，移动应用，物联网，无人机，大数据预测模型，并且在这些软件系统中检测到authentication bug, memory corruption, cryptographic misuses, OTP misuses, input validation bug等。针对于现有已检测的漏洞，马老师团队利用软件工程领域的软件分析方法 （e.g., control flow analysis, data flow analysis）对当前漏洞实现进行代码检测。通过学习现有bug 代码的生成模式来给出对应的修复方案，同时利用机器学习的方法对此类修复方案进行模式学习来实现自动化漏洞的修复（e.g., CDRep, VuRLE）。


目前马老师诸多方向都有深入研究和扩展，包含: 

  • security analysis on mobile apps, IoT firmwares, UAV systems; 

  • backdoor analysis

  • private privacy


马老师团队隶属于昆士兰大学信息科学技术与电子工程学院。昆士兰大学始建于1909年，是一所老牌著名研究型大学，是昆士兰州第一所综合型大学，也是澳大利亚最大、最有声望的大学之一。根据QS2021 ranking，其在全球排名位列第46位。



昆士兰大学的St Lucia主校区位于昆士兰州布里斯班市。布里斯班通常被人们认为是一座“澳大利亚最适合居住的城市”，为那里的居民和游客提供了一个安全、多文化、友好的生活环境。拥有180多万人口的布里斯班市具有国际化大都市的气息和世界一流的现代化设施，如世界级的国际机场、文化中心、会议中心、博物馆和艺术陈列馆等等。





马思奇老师的团队热烈欢迎有志博士生，访问学者，暑期实习生。昆士兰大学开放博士生奖学金计划来资助优秀的本科和硕士毕业生，同时还有各类其他奖学金可供申请（e.g., Data61 Scholarship）。欢迎各位感兴趣的同学欢迎同马老师联系 （Email：slivia.ma@uq.edu.au）呀~
